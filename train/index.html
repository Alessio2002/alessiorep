1. As a user I would like to open the app and be able to select the departure and arrival in order to see the scheduled trains.
  2. As a user I would like for the app to be as clear as possible in order to navigate it easily.
  3. As a user I would like to be able to click on a scheduled train of my liking in order to buy a train ticket.
  4. As a user I would like to receive a confirmation message after my purchase in order to see the details of my trip clearly.
  5. As a user I would like to receive a message if a disruption was to happen in order to be assured if the train happens to be late or cancelled.
